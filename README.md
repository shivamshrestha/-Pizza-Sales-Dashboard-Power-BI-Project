# 🍕 Pizza Sales Dashboard – Power BI Project

## 📊 Overview

This project focuses on analyzing pizza sales data using Power BI to generate meaningful business insights.

## 📌 Key Metrics

* Total Revenue: 817.86K
* Total Orders: 21.35K
* Total Pizzas Sold: 49.57K
* Average Order Value: 38.31
* Avg Pizza per Order: 2.32

## 🔍 Key Insights

* The Classic and Thai Chicken pizzas are top performers across revenue, quantity, and orders
* Sales peak during weekends, especially Friday and Saturday evenings
* Highest sales months are July and January
* Bottom-performing pizzas contribute minimal sales and need improvement

## 🛠 Tools Used

* Power BI
* DAX
* Excel

## 🚀 Features

* Interactive dashboard with filters
* KPI cards for quick insights
* Top & Bottom product analysis
* Time-based trend analysis
* Clean and user-friendly design

## 📈 Learning Outcomes

* Data cleaning and transformation
* Data modeling (Star Schema)
* Writing DAX measures
* Building interactive dashboards
* Generating business insights
